## Agent skills

### Issue tracker

Issues are tracked as local markdown files under `.scratch/<feature>/` (no remote tracker). See `docs/agents/issue-tracker.md`.

### Triage labels

Default triage vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix), carried in each issue's `status:` frontmatter. See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: `CONTEXT-MAP.md` at root points to per-crate `CONTEXT.md` files. See `docs/agents/domain.md`.
