# CR1140 Native Rust HAL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a reusable Rust HAL (`cr1140-hal`) and tracer-bullet apps for the ifm CR1140/CR1141 (aarch64 / i.MX 8M Nano, Yocto Linux), deployable in place of the stock CODESYS runtime.

**Architecture:** A single Cargo workspace with one library crate (`cr1140-hal`) exposing four modules — `display`, `input`, `can`, `sys` — over stock Linux ABIs (fbdev/DRM, evdev, SocketCAN, sysfs). Pure-Rust deps only (`nix`, `socketcan`, `bitflags`). Host-testable logic (pixel addressing, event decoding, sysfs value parsing) is built TDD-first against in-memory buffers; hardware-binding layers are thin wrappers verified by on-device smoke tests. Cross-compilation uses `cargo-zigbuild` (zig as linker/sysroot) so we can target both `aarch64-unknown-linux-musl` (default, static) and a pinned-glibc `aarch64-unknown-linux-gnu` from macOS without Docker.

**Tech Stack:** Rust (stable), `cargo-zigbuild` + `zig`, `nix`, `socketcan`, `just`. Target device reached over SSH at `192.168.1.102`.

## Reference: Confirmed device facts (from delivery, pre-recon)

- SoC: NXP i.MX 8M Nano → **aarch64** (Cortex-A53).
- eMMC `/dev/mmcblk0`: `p1` rootfs (bootable, RO+overlay), `p2` ext4, `p3` large ext4 **data partition** (writable, our deploy target).
- Boot: `init=/sbin/ifm-overlay.sh`; kernel args `ifm_boot_backlight`, `ifm_orientation`, `ifm_boot_status_led`.
- SSH: `192.168.1.102`, default username/password (record exact values in `docs/device-facts.md`, do not commit secrets).

## Reference: input_event / can_frame layout (aarch64, 64-bit)

- `struct input_event` = `timeval{tv_sec: i64, tv_usec: i64}` (16 B) + `type: u16` + `code: u16` + `value: i32` = **24 bytes**.
- `EV_KEY` = 1, `EV_SYN` = 0. Key `value`: 0=release, 1=press, 2=autorepeat.
- `struct can_frame` (classic) = `can_id: u32` + `can_dlc: u8` + 3 pad + `data: [u8;8]` = 16 bytes (handled by the `socketcan` crate).

/// A mutable view over a packed pixel buffer (xRGB8888, 4 bytes/pixel).
pub struct Surface<'a> {
    pub buf: &'a mut [u8],
    pub width: u32,
    pub height: u32,
    pub stride: u32, // bytes per row (may exceed width*4)
}

impl<'a> Surface<'a> {
    pub fn new(buf: &'a mut [u8], width: u32, height: u32, stride: u32) -> Self {
        Self { buf, width, height, stride }
    }

    /// Byte offset of pixel (x, y); None if out of bounds.
    pub fn offset(&self, x: u32, y: u32) -> Option<usize> {
        if x >= self.width || y >= self.height { return None; }
        Some((y * self.stride + x * 4) as usize)
    }

    pub fn put_pixel(&mut self, x: u32, y: u32, color: u32) {
        if let Some(o) = self.offset(x, y) {
            self.buf[o..o + 4].copy_from_slice(&color.to_le_bytes());
        }
    }

    pub fn fill(&mut self, color: u32) {
        for y in 0..self.height {
            for x in 0..self.width {
                self.put_pixel(x, y, color);
            }
        }
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn offset_respects_stride_and_bounds() {
        let mut buf = vec![0u8; 16 * 4 * 4]; // stride 64, 4 rows... oversize
        let s = Surface::new(&mut buf, 4, 4, 64);
        assert_eq!(s.offset(0, 0), Some(0));
        assert_eq!(s.offset(1, 0), Some(4));
        assert_eq!(s.offset(0, 1), Some(64));
        assert_eq!(s.offset(4, 0), None); // x out of bounds
        assert_eq!(s.offset(0, 4), None); // y out of bounds
    }

    #[test]
    fn put_pixel_writes_le_xrgb() {
        let mut buf = vec![0u8; 4 * 4 * 4];
        let mut s = Surface::new(&mut buf, 4, 4, 16);
        s.put_pixel(1, 0, 0x00FF8040);
        assert_eq!(&buf[4..8], &[0x40, 0x80, 0xFF, 0x00]);
    }

    #[test]
    fn fill_sets_every_pixel() {
        let mut buf = vec![0u8; 2 * 2 * 4];
        let mut s = Surface::new(&mut buf, 2, 2, 8);
        s.fill(0x11223344);

use nix::sys::mman::{mmap, MapFlags, ProtFlags};
use std::fs::OpenOptions;
use std::os::fd::AsRawFd;
use std::num::NonZeroUsize;

// Linux fb ioctls (from <linux/fb.h>)
const FBIOGET_VSCREENINFO: u64 = 0x4600;
const FBIOGET_FSCREENINFO: u64 = 0x4602;

#[repr(C)]
#[derive(Default, Clone, Copy)]
struct FbBitfield { offset: u32, length: u32, msb_right: u32 }

#[repr(C)]
#[derive(Default, Clone, Copy)]
struct FbVarScreeninfo {
    xres: u32, yres: u32, xres_virtual: u32, yres_virtual: u32,
    xoffset: u32, yoffset: u32, bits_per_pixel: u32, grayscale: u32,
    red: FbBitfield, green: FbBitfield, blue: FbBitfield, transp: FbBitfield,
    nonstd: u32, activate: u32, height: u32, width: u32, accel_flags: u32,
    pixclock: u32, left_margin: u32, right_margin: u32, upper_margin: u32,
    lower_margin: u32, hsync_len: u32, vsync_len: u32, sync: u32,
    vmode: u32, rotate: u32, colorspace: u32, reserved: [u32; 4],
}

#[repr(C)]
struct FbFixScreeninfo {
    id: [u8; 16], smem_start: u64, smem_len: u32, type_: u32, type_aux: u32,
    visual: u32, xpanstep: u16, ypanstep: u16, ywrapstep: u16,
    line_length: u32, mmio_start: u64, mmio_len: u32, accel: u32,
    capabilities: u16, reserved: [u16; 2],
}

pub struct FbDisplay {
    map: *mut u8,
    len: usize,
    pub width: u32,
    pub height: u32,
    pub stride: u32,
}

impl FbDisplay {
    /// Open a framebuffer device (e.g. "/dev/fb0").
    pub fn open(path: &str) -> std::io::Result<Self> {
        let file = OpenOptions::new().read(true).write(true).open(path)?;
        let fd = file.as_raw_fd();
        let mut var = FbVarScreeninfo::default();
        let mut fix: FbFixScreeninfo = unsafe { std::mem::zeroed() };
        unsafe {
            if libc_ioctl(fd, FBIOGET_VSCREENINFO, &mut var as *mut _ as *mut _) != 0 {
                return Err(std::io::Error::last_os_error());
            }
            if libc_ioctl(fd, FBIOGET_FSCREENINFO, &mut fix as *mut _ as *mut _) != 0 {
                return Err(std::io::Error::last_os_error());
            }
        }
        let len = fix.line_length as usize * var.yres as usize;
        let map = unsafe {
            mmap(None, NonZeroUsize::new(len).unwrap(),
                 ProtFlags::PROT_READ | ProtFlags::PROT_WRITE,
                 MapFlags::MAP_SHARED, &file, 0)?
        };
        // keep file alive for the life of the mapping
        std::mem::forget(file);
        Ok(Self { map: map as *mut u8, len, width: var.xres, height: var.yres, stride: fix.line_length })
    }

    /// Borrow the framebuffer as a Surface for drawing.
    pub fn surface(&mut self) -> Surface<'_> {
        let buf = unsafe { std::slice::from_raw_parts_mut(self.map, self.len) };
        Surface::new(buf, self.width, self.height, self.stride)
    }
}

extern "C" {
    #[link_name = "ioctl"]
    fn libc_ioctl(fd: i32, request: u64, arg: *mut core::ffi::c_void) -> i32;
}

pub const EV_KEY: u16 = 1;

/// Raw evdev event (aarch64 layout = 24 bytes).
#[derive(Debug, PartialEq, Eq, Clone, Copy)]
pub struct InputEvent {
    pub type_: u16,
    pub code: u16,
    pub value: i32,
}

impl InputEvent {
    pub const SIZE: usize = 24;

    /// Decode one 24-byte evdev record. Returns None if the slice is too short.
    pub fn decode(bytes: &[u8]) -> Option<InputEvent> {
        if bytes.len() < Self::SIZE { return None; }
        // bytes 0..16 = timeval (ignored); 16..18 type; 18..20 code; 20..24 value
        let type_ = u16::from_le_bytes([bytes[16], bytes[17]]);
        let code = u16::from_le_bytes([bytes[18], bytes[19]]);
        let value = i32::from_le_bytes([bytes[20], bytes[21], bytes[22], bytes[23]]);
        Some(InputEvent { type_, code, value })
    }
}

/// Translate a raw event into a logical ButtonEvent (press/release only).
pub fn to_button_event(ev: InputEvent) -> Option<ButtonEvent> {
    if ev.type_ != EV_KEY { return None; }

        0 => Some(ButtonEvent::Released(btn)),

```rust
//! CAN access via SocketCAN.
use socketcan::{CanSocket, CanFrame, Socket, EmbeddedFrame, StandardId, Id};

pub struct CanBus { sock: CanSocket }

impl CanBus {
    /// Open an existing SocketCAN interface (e.g. "can0"). The interface must
    /// already be up (`ip link set can0 up type can bitrate 250000`).
    pub fn open(iface: &str) -> std::io::Result<Self> {
        let sock = CanSocket::open(iface)?;
        Ok(Self { sock })
    }

    pub fn send_std(&self, id: u16, data: &[u8]) -> std::io::Result<()> {
        let sid = StandardId::new(id).ok_or_else(|| std::io::Error::new(
            std::io::ErrorKind::InvalidInput, "id out of 11-bit range"))?;
        let frame = CanFrame::new(Id::Standard(sid), data).ok_or_else(|| std::io::Error::new(
            std::io::ErrorKind::InvalidInput, "data too long"))?;
        self.sock.write_frame(&frame).map_err(std::io::Error::from)
    }

    /// Blocking receive of the next frame; returns (id, data bytes).
    pub fn recv(&self) -> std::io::Result<(u32, Vec<u8>)> {
        let frame = self.sock.read_frame().map_err(std::io::Error::from)?;
        let id = match frame.id() {
            Id::Standard(s) => s.as_raw() as u32,
            Id::Extended(e) => e.as_raw(),
        };
        Ok((id, frame.data().to_vec()))
    }
}
```
